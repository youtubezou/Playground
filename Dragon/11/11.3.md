## 11.3 Iteration Spaces

### 11.3.1

> Convert each of the following loops to a form where the loop indexes are each incremented by 1:

> a) `for (i=10; i<50; i=i+7) X[i, i+1] = 0;`.

```
for (j = 0; j <= 5; j++)
    X[7*j+10, 7*j+11] = 0;
```

> b) `for (i= -3; i<=10; i=i+2) X[i] = X[i+1];`.

```
for (j = 0; j <= 6; j++)
    X[2*j-3] = X[2*j-2];
```

> c) `for (i=50; i>=10; i--) X[i] = 0;`.

```
for (j = 0; j <= 40; j++)
    X[50-j] = 0;
```
