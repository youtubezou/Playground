## 2.4 Parsing

### 2.4.1

> Construct recursive-descent parsers, starting with the following grammars:

> a) $$S ~\rightarrow~ +~S~S~|~-~S~S~|~\mathbf{a}$$

```c
void S() {
    switch (lookahead) {
    case '+':
        match('+'); S(); S(); break;
    case '-':
        match('-'); S(); S(); break;
    case 'a':
        match('a');
    default:
        report("syntax error");
    }
}

void match(terminal t) {
    if (lookahead == t) {
        lookahead = nextTerminal;
    } else {
        report("syntax error");
    }
}
```
