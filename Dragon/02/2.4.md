## 2.4 Parsing

### 2.4.1

> Construct recursive-descent parsers, starting with the following grammars:

> a) $$S ~\rightarrow~ +~S~S~|~-~S~S~|~\mathbf{a}$$

```c
void S() {
    switch (lookahead) {
    case '+':
        match('+'); S(); S(); break;
    case '-':
        match('-'); S(); S(); break;
    case 'a':
        match('a');
    default:
        report("syntax error");
    }
}

void match(terminal t) {
    if (lookahead == t) {
        lookahead = nextTerminal;
    } else {
        report("syntax error");
    }
}
```

> b) $$S ~\rightarrow~ S~(~S~)~S~|~\epsilon$$

```c
void S() {
    if (lookahead == '(') {
        match('('); S(); match(')'); S();
    } else {
        report("syntax error");
    }
}
```

> c) $$S ~\rightarrow~ 0~S~1~|~0~1$$

```c
void S() {
    match('0');
    if (lookahead == '1') {
        match('1');
    } else {
        S(); match('1');
    }
}
```
