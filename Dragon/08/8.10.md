### 8.10.2

> Generate optimal code using two registers for each of the expressions of Exercise 8.10.1.

> a) `a/(b+c)-d*(e+f)`.

```
LD  R2, f
LD  R1, e
ADD R2, R1, R2
LD  R1, d
MUL R2, R1, R2
ST  t4, R2
LD  R2, c
LD  R1, b
ADD R2, R1, R2
LD  R1, a
DIV R2, R1, R2
LD  R1, t4
SUB R2, R2, R1
```

> b) `a+b*(c*(d+e))`.

```
LD  R2, e
LD  R1, d
MUL R2, R1, R2
LD  R1, c
MUL R2, R1, R2
LD  R1, b
MUL R2, R1, R2
LD  R1, a
ADD R2, R1, R2
```

### 8.10.3

> Generate optimal code using three registers for each of the expressions of Exercise 8.10.1.

> a) `a/(b+c)-d*(e+f)`.

```
LD  R3, f
LD  R2, e
ADD R3, R2, R3
LD  R2, d
MUL R3, R2, R3
LD  R2, c
LD  R1, b
ADD R2, R1, R2
LD  R1, a
DIV R2, R1, R2
SUB R3, R2, R3
```

> b) `a+b*(c*(d+e))`.

```
LD  R2, e
LD  R1, d
MUL R2, R1, R2
LD  R1, c
MUL R2, R1, R2
LD  R1, b
MUL R2, R1, R2
LD  R1, a
ADD R2, R1, R2
```

> c) `(-a+*p)*((b-*q)/(-c+*r))`.

```
LD  R3, r
ST  R3, *R3
LD  R2, c
SUB R2, #0, R2
ADD R3, R2, R3
LD  R2, q
ST  R2, *R2
LD  R1, b
SUB R2, R1, R2
DIV R3, R2, R3
LD  R2, p
ST  R2, *R2
LD  R1, a
SUB R1, #0, R1
ADD R2, R1, R2
MUL R3, R2, R3
```
