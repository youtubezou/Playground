> Exercise 1.2 [\**] What sets are defined by the following pairs of rules? Explain why.
> 1. (0, 1) ∈ S
>    (n, k) ∈ S
>    (n+1, k +7) ∈ S
> 2. (0, 1) ∈ S
>    (n, k) ∈ S
>    (n+1, 2k) ∈ S
> 3. (0, 0, 1) ∈ S
>    (n, i, j) ∈ S
>    (n +1, j, i+ j) ∈ S
> 4. [\***] (0, 1, 0) ∈ S
>    (n, i, j) ∈ S
>    (n +1, i +2, i + j) ∈ S

1. { (n, 7n + 1) | n ∈ N }.

2. { (n, 2 ^ n) | n ∈ N }.

3. { (n, fib(n), fib(n + 1)) | n ∈ N }, where fib(i) is the ith fibonacci number start from 0.

4. { (n, 2n + 1, n^2) | n ∈ N }.
